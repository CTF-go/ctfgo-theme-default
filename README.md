# CTFgo-Theme-Default

## Frontend of CTFgo

It is the default theme of CTFgo, whitch is written in vue.

## Tech stack

vue2, vue-router, webpack, Stylus, Vuesax

## Usage

### For Developer

```
git clone https://github.com/CTF-go/ctfgo-theme-default.git

cd ctfgo-theme-default

yarn install

npm run serve
```
